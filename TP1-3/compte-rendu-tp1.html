Exercice 2 : Document HTML de base

À quoi servent les tags h1, h2 et h3 ?

Les tags 'h1', 'h2' servent à écrire des textes avec une police d'écriture de plus en plus petite(sous forme de titre). La balise h1 sera plus imposante qu'une balise h3

À quoi sert le tag p ?

Les tags 'p' servent à écrire du texte sous forme de paragraphe

À quoi correspondent les 3 valeurs sur la première ligne de l'en-tête ?

La première valeur correspond à la requête

La deuxième valeur corresepond au code status

La troisième valeur est la description du code status

Quel est le nom du serveur Web ?

À quoi correspond la valeur de Content-Type ?

La valeur Content-Type correspond au type de donnée renvoyée

À quoi correspond la valeur de Content-Length ?

La valeur Content-Length correspond a la longueur du document
